## Installation

Frost requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and devDependencies and start the server.

```sh
git clone https://github.com/Frost-sa/forst-nuxtjs2.git
cd forst-nuxtjs2
yarn
yarn dev
```

For production ...

```sh
yarn build
```
